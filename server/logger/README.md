# Logger

## Quill

We use the [Quill Asynchronous Low Latency C++ Logging Library](https://github.com/odygrd/quill)
for the Unid Project Logger

### Installation and usage

Quill v11.1.0 is installed using the Conan C++ library manager.

Usage follows the project's recommended usage because of it's high use of efficient macros, and
overall performant architecture.
[Recommended usage documentation](https://github.com/odygrd/quill/tree/master/examples/recommended_usage).
